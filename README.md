# cdav
Console WebDAV client for Linux.

cdav is currently in its initial development phase.

### Currently working on:
- Commandfile

### Pending:
- Formatted output
- GUI (cdav-gtk)

### Ideas:
- Implement CardDAV
- Implement CalDAV

## Compile

To build cdav in its current state, make sure to run 
`make deps` before running `make`.
